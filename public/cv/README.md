# CV Download Instructions

## How to Add Your CV

1. **Prepare Your CV**: Create your CV/Resume as a PDF file
2. **Naming**: Name it exactly: `Faizan-Pervez-CV.pdf`
3. **Location**: Place it in this folder (`/public/cv/`)
4. **Done**: The download button will work immediately!
